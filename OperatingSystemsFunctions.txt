Security-Operating systems use a variety of methods to ensure user security. 
These include passwords, facial recogntion, fingerprint recognition, and swiping areas of the screen in a specific order.
Beyond sign in security, operating systems can offer users firewall and anti virus software to protect their information stored on the device.
Memory Managment-The operating system manages the computers main memory. Main memory is storage that can be accessed directly through the CPU. The operating system keeps track of the primary memory and allocates the memory to a process when the process requests it and deallocates the memory when the process has terminated or is performing an operation. 
File Management-An operating system can oragnize files into a directory for a user, which makes navigation more efficient. It will carry out the following file management activities: where information is stored, user access settings and status of every file. These activities are known as the file system
